
# Minesweeper Game

## Overview

A classic Minesweeper game implementation using HTML, CSS, and JavaScript. The game allows players to select a difficulty level, reveals cells, flags potential mines, and tracks game time and remaining mines.

## Features

*   **Difficulty Levels:** Easy, Medium, and Hard options that adjust the grid size and number of mines.
*   **Game Board:** A dynamic grid that adapts to the selected difficulty.
*   **Cell Interaction:**
    *   Click to reveal a cell.
    *   Right-click to flag a cell.
*   **Game Status:**
    *   Timer to track the duration of the game.
    *   Mine counter to show the number of remaining mines.
*   **Responsive Design:** The game board and controls are designed to work on different screen sizes.

## Project Structure

*   `index.html`: The main HTML file containing the structure of the game.
*   `style.css`: The CSS file for styling the game.
*   `main.js`: The JavaScript file containing the game logic.
*   `blueprint.md`: This file, documenting the project.
